### Comandos básicos

- COMANDO para criar a migração
```
  python manage.py makemigrations people
```  

- COMANDO para executar a migração:
```
  python manage.py migrate people
```  

- COMANDO para subir o servidor na porta 8000:
```
  python manage.py runserver
```
